Introduction

In this documentation, you will find the basic installation, properties and special functions to control the plugin.
 
 
It's hard to explain with words, and thats why I created 10 different tutorials that will show you how to use the plugin and functions.

Get Started

 

Add this in your head section
 
 
<script src="static/js/jquery-1.11.1.min.js"></script>
IMPORTANT!
This is the standard jQuery library, you don't need to import it if you are using other version.
 
 
 
<!-- crystalnotifications CSS (needed) -->
<link rel="stylesheet" type="text/css" href="static/css/crystalnotifications.min.css">
Add the Crystal Notification CSS (the style). I recommend you to use the minify version. because is much lighter and more secure.
 
 
 
 
<!-- crystalnotifications JS (needed) -->
<script src="static/js/crystalnotifications.min.js"></script>
This files contains all the functions and calculations. This is required to make it work.
 
 
 
 
 
 

Crystal Notification Properties

Property Default Optional Description
position 0 1,2,3,4 and 0 Crystal notification position.
width 300 450 Max-width of the Crystal Notification
title Crystal Notification title "Any string" You can add any kind of string here, this is going to be the group title on the Crystal Notification Center
content "" Any kind of string or small HTML fragment  
image undefined "path/of/image" Image path
sound true false Enabled or disabled the sound
panelbutton true false If you want to show the white line that show the notification center
closebutton true false Show or hide the Close icon.
smallcloseclick true false Enabled or disabled, close on click event
timeout undefined 3000 3000 = 3 seconds, 5000 = 5 seconds
addtopanel true false Add to the notification center.
soundpath "static/sound" "path/of/sound" If you are using a different path.

CrystalNotificationPanel

Property Default Optional Description
title "Information" "Any String" This is the main group title
image undefined "path/of/image" Image URL
content "" Small HTML fragment or any text This is the notification center content
show true false This is not required for anything, I use this property to control some stuff.
showpanel false true Show panel right after you call it.
There are a couple of ways to show the notification center, this is one of them.

Special Functions

This functions will show you how to change the behavior of this plugin.

 

Check the tutorials and you will have a better idea on how to use them.

 

 

NukeAllTheNotifications()

Remove all the items inside the Notification Center and will show the Empty Label. 

If you want to change this label, simply open the CrystalNotification.js, modify line 17:

var cnEmptyMessage = "You don't have any notifications.";

 

CloseAllCrystals()

Close all the Crystal Notifications in the visible area. (Callback functions won't be perform unless has timeout)

 

CloseCrystalNotificationPanel()

Close the notification Panel

 

ShowCrystalNotificationPanel()

Show the Cotification Center

Tutorial #1

"1. Launching on form load"

This tutorial will fire a Crystal Notification on form load with some properties. You can add this code anywhere inside  your <body> tag
 
<script type="text/javascript">

$(document).ready(function(){

$.CrystalNotification({

title: "Hello!",

image: "static/img/Colorfull/Messages-colorfull.png",

content: "Ready to go!",

});

});

</script>
 

 

 

Tutorial #2

2. Launching Notification Center on load

Similar to the previous tutorial, but this call a special function to show/hide the Notification Panel

 

 

<script type="text/javascript">

$(document).ready(function(){

// Just type this on a javascript block

ShowCrystalNotificationPanel();

});

</script>

 

Tutorial #3

3. Show Panel on button click

This tutorial will help you to launch the panel on a button click.

You can set this code anywhere inside the <body> tag

 

 
<script type="text/javascript">

//jQuery button click function
$("#TestButton").on("click",function(){


// Just type this on a javascript block
ShowCrystalNotificationPanel();


});

</script>

Tutorial #4

4. Call Crystal Notification with all the parameters

This tutorial will help you to understand how to change any property.

 

 

 

$("#TestButton").on("click",function(){


$.CrystalNotification({

title: "This is my title",   // Title

image: "static/img/Colorfull/Messages-colorfull.png", // Image

content: "Write some content...", // Content

sound: true, // If you want to disabled or enabled the sound

panelbutton: true, // If you want to show or not the button line that shows the Notification Center

closebutton: true, // If you want to hide the Close button

timeout: 4000, // Timeout to autoclose and execute a callback if is set

addtopanel: true, // if you don't want to display it on the notification center

});


});

 

Tutorial #5

5. Crystal Notification with callback

Callbacks are ways to know when the user closes or clock in the panel.

With this, you can perform an JavaScript action.

In this tutorial, you will see how fire an alert after you close the Crystal Notification.

 

 

$("#TestButton").on("click",function(){

$.CrystalNotification({

title: "This is my title", 

image: "static/img/Colorfull/Messages-colorfull.png",

content: "Write some content...",

panelbutton: false,

},function(){

alert("Callback!");

});

});

Tutorial #7



	7. Notification Center Callback


	Notification Center, can manage Callbacks too. Sometimes you need to show something when your user clicks on one of the options.


	 


	This is the way to do it.


	 


	This tutorial will help you to understand  this behavior. When you press an option, an Alert will be fire showing the Title Name.


	 


	 


$(document).ready(function(){


$.CrystalNotificationPanel({
title: "iMessage", 
image: "static/img/Colorfull/Messages-colorfull.png",
content: "Write some content..."
},function(){
alert("iMessage");
});


$.CrystalNotificationPanel({
title: "Gmail", 
image: "static/img/Colorfull/Gmail-colorfull.png",
content: "Write some content..."
},function(){
alert("Gmail");
});


//Show the panel
ShowCrystalNotificationPanel();


});





	
Tutorial #8



	8. Notification Center Custom HTML Content



	Crystal Notification and Crystal Notification Center, can store custom HTML content. just create a string variable with all the HTML content and that's it. Just remember to don't break the string with the ( ' ) or ( " )



	 



	 


$(document).ready(function(){

// This is a custom HTML that shows Google Maps
$.CrystalNotificationPanel({
title: "Google Maps",
image: "static/img/Colorfull/Google-Maps-128.png",
content: '<iframe width="100%" height="400px" frameborder="0" scrolling="no" marginheight="0" marginwidth="0" src="https://maps.google.es/maps?f=q&amp;source=s_q&amp;hl=es&amp;geocode=&amp;q=san+jose+costa+rica&amp;aq=&amp;sll=43.281204,-7.921143&amp;sspn=2.847262,5.811768&amp;ie=UTF8&amp;hq=&amp;hnear=San+Jos%C3%A9,+Costa+Rica&amp;ll=9.927128,-84.082012&amp;spn=0.060197,0.090809&amp;t=m&amp;z=14&amp;output=embed"></iframe>',
}); 

// A youtube video.
$.CrystalNotificationPanel({
title: "YouTube", 
image: "static/img/Colorfull/YouTube-128.png",
content: '<iframe width="100%" height="315" src="http://www.youtube.com/embed/5EjG-1U3wqA" frameborder="0" allowfullscreen></iframe>'
});



//Show the panel
ShowCrystalNotificationPanel();


});
