Installation

Install is pretty simple:

 

1 -  Copy the static folder inside "3. Main Files", that contains all you need to run the plugin.

 

2 - Add this references to your page:

jQuery: Is not required if you are using another jQuery library, (we need 1.4 or newer)

<script src="static/js/jquery-2.1.4.min.js"></script>

 

Greensock: Is a gsap compilation of: 

  1. CSSPlugin
  2. EasePack
  3. TimelineMax
  4. TweenLite
 
KL-Animate.js: Is the code that makes the magic flow.
<script src="static/js/greensock.js"></script>
<script src="static/js/kl-animate.min.js"></script>

 

Notes:

1- Make sure you import the KL-Animate.js after jQuery. Can be placed on bottom or in the <head> section.

2- Remember to call the plugin when all the elements you want to animate, are created on the page... I mean, first create the elements and then call the initializator. 

3- You can use "document ready method", but is better if you call the plugin just after the last element is created, so, the animation will be ready by that moment.

4- I recommend you to put the klanimate, if not, add this line in some css file  .klanimate{visibility:hidden}, just to make sure all the elements that you want to animate will be hidden before page load.

 

 

Basic Explanation

KL-Animate will create enter animation or change the position of an element, you can even add animations to anything that are currently in action.

 

First create your page as you wanted, then add the animations. Remember, this will not alter your html structure or should not create strange things on your page.

 

KL-Animate works by adding a simple class, that the only thing it does, is to hide the element until the moment is right and let the code know what element animate.

 

Configuring the animation is simple too, you need to add data-attributes to an element with the klanimate class.

 

Important:

* There are 11 tutorials to show you how the attributes work, but I will explain each one in here.

* If you are reviewing this documentation before purchasing the code, those tutorials are linked in the demosite and here.

Basic HTML Structure

Here's and example of how this works:

Just the body part.

<body>

<div class="container">


<h1 class="klanimate">Hello World! <small>scroll down</small></h1>


<h1 class="klanimate">Hi! <small>keep scrolling</small></h1>


<h1 class="klanimate">This is the end!</h1>


<span class="klanimate">boo!</span>


</div>

<script>

$.klAnimate();

</script>



</body>

 

This code, create a fadeIn animation when the element is on the browser viewpoint.

Properties

The following propertyes can be change using data-attributes, for example:

<span class="klanimate" data-animation = "elastic" data-x="100">  This is a span </span>

 

Time properties, are all in SECONDS. 

 

Property Default value Optional Description
animation fadeIn
  • elastic
  • bounce
  • slowmo
  • circle
  • back
 This animations will be visible only when the HTML element has some movement.
delay 0 Any number in seconds This will make a delay in 'X' amount of seconds after the element is on the screen
rotation 0 Any positive or negative value Positive values will rotate clockwise and negative counter clockwise
visible 0  1 or 0 This will prevent the fadeIn animation, in other words, the element will be always visible if you set this attribute equals one.
scale 1 Any positive value This property create a scale animation, negative values create a strange thing, so I recommend you to use only positive ones.
duration 1 Any positive value If you want a slower animation, just put a higher value, for a quicker animation just a lower one
x 0 Any positive or negative value This is a number of how many pixels the HTML element will move horizontally
y 0 Any positive or negative values This is a number of how many pixels the html element will move vertically
repeat 0

-1 = Infinite Loop

0 = No repeat

Any other number will create that number of sequences

 This controls the repeat functionality.

 

Animating Numbers

To animate numbers, there are a couple of properties and you use it like this:

 

<h1 class="klanimate" data-initial="100" data-number="500" data-timer="7" data-color="red"> 0 </h1>

 

Property Value Description
data-initial (optional value) Some number This is the starting point, it is optional, if you don't set it, it will begin on Zero
data-number Required - Some number higher than data-initial value Goal number.
data-timer Optional - positive number How much time do you need to achieve the "data-number" value
data-color Optional - Some HTML color This will change the color of the increasing number

Check the tutorial 11 to see this in action.
