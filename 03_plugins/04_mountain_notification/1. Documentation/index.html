Installation instructions

Put this in your <head> section

 

jQuery library import, remember, you don't need to RE-IMPORT this library if there is other jquery version. This plugin shoud work fine from jQuery 1.7

<script src="static/js/jquery-1.9.0.min.js"></script>

 

jQuery UI, this one is required for dragging functionality, if you want to remove it, you will need to delete any "drag" reference inside "mountainnotifications.js" file.

<script src="static/js/jquery-ui.1.10.3.min.js"></script>

 

mountainnotifications.css, this contains the Style of all the plugins and the css animations.

<link rel="stylesheet" type="text/css" href="static/css/mountainnotifications.css">

 

mountainnotifications.js, this one, contains all the jQuery functions and definitions

<script src="static/js/mountainnotifications.js"></script>

 

NOTE:

I include the min files, if you are going to a development environment, I recommend to use this files, are smaller and harder to read

 

If you want a free site to minimize your files:

http://htmlcompressor.com/compressor/

 

 

 

 

Global Properties

Global Properties are some variables that exist in the entire page.

 

You can make changes to those properties in "mountainnotifications.js" file.  (Line 16)

 

 

Here's a description of every one:

1 - NotificationBarEmptyText:  This is the text that will appear inside the Notification Panel when has nothing to display

 

2 - ESCclosesNotificationBar: If you dont want to close on ESC Keypress, set this property to false

3 - CloseNotificationOnClick: Close the notification bar by clicking  outside of the notification bar

4 - OpenNotificationWithESC: If the notification bar is closed, you can open it with ESC Key

5 - LabelMinimize: Minimize's button tooltip

6 - LabelMaximize: Maximize's button tooltip

7 - LabelClose: Close's button tooltip

 

 

Notification Panel

Notification Panel, is a sidepanel like Mountain Lion notification center.
 
 

 

Properties:

1 - title: Notification's title text
 
2 - img: Notification's image URL
 
3 - clearbutton: True or False value, this one makes appear or dissapear the CROSS button that remove that section.
 
4 - items: this is an array of items, every Item will appear under the title. Every item, has properties and the capability to store a callback function.
  • title: item's title
  • date: This is a string that I use it to show when the notification is called
  • content: The main content of the notification item, can store HTML code
  • callback: This is a function that will be called, when the user touch or click this section

For better understanding, go to the tutorial folder and watch it in action.

 

Modal PopUp

ModalPopUp is a small fragment of HTML or iFrame that appear in the middle of the screen.

 

Properties

1- html: This is a String that can hold a HTML. Remember, if you are going to add an iFrame, its better to use the iFrame property instead of adding in here. 
 
2- width: This is a String that holds the width of the modal. If the user window width is smaller than this, it will take 98% of the user view.
 
3- height: This is a String that holds the height of the modal. If the user window height is smaller than this, it will take 98% of the user view.
 
4- blackscreen: true or false, this disabled the background and only Modals can be clicked
 
5- title: This is a text that holds the modal's title, you wont see this title unless you open or use the notification bar property.
 
6- content:  This is a text that will appear in the notification bar if is used.
 
7- img:  This is the image that will appear in the notification bar if is used.
 
8- notificationbar: true or false, this tells the plugin, that you want to display it in the notification bar too.
 
9- close: true or false, with this you enable or disabled the upper left buttons items.
 
10- iframe:  URL of the iframe that you want to display, leave this field empty if you want to use the HTML content.
 
11- iframescrolling: true or false, enabled or disabled the iFrame scrolling function
 
12- draggable: true or false, enable or disable the dragging functionallity
 
13- animation: this is the name of animation. see "Animations" section to see the available ones
timeout: undefined

 

 

 

Small Notifications

 

Small notification in the upper right side of the window. or Middle up for small screens.

 

Properties

1- title: (optional) notification's title
 
2- content: notification's content, can store HTML code
 
3- img: the image of the small notification.
 
4- buttons: (optional) if you want buttons on the small notification. See the tutorial folder for more information about this.
 
5- closeonclick: true or false, set this to true, if you want to close the notification when is touch or clicked.
 
6- timeout: Time in miliseconds before disappear.
 
7- notificationbar: true or false, set this value to true, if you want to include this into the notification bar.
 
8- animation: this change the IN animation. (See Animation section for more details)

 

Banners

Upper or bottom information, with maximize button.

 

 

Properties

1- position: "top or "bottom", Banner's position
 
2- title: Banner's title
 
3- content: Banner's content, can store HTML content
 
4- img: Banner's image
 
5- timeout: time in miliseconds to autohide the banner
 
6- animation: In animation (see animation section for more details).
 
7- notificationbar: If you want to display this in the notification bar
 
8- withbuttons: true or false, shows or hide the button section.
 
 

 

MessageBox

Obtain information from your user with style

 

Properties

1- withheader: true or false, shows or hide the messagebox upper bar
 
2- headertext: This is a text that will appear in the header section. Only visible if "withheader" is true
 
3- width: Messagebox width
 
4- title: Messagebox title
 
5- content: Messagebox content
 
6- img: Messagebox's URL
 
7- timeout: Time in miliseconds, if you want to autoclose the notification after some time.
 
8- draggable: true or false, this makes the notification draggable
 
9- buttons: Text in some format that tells the plugin the number of buttons you need
 
10- animation: Messagebox In Animation
 
 
 

Login

Need a login on the go?

 

Properties

1- withheader: true or false, if you want to display a header bar
 
2- headertext: header's bar text, only appear if the "withheader" is true
 
3- width: Login's width
 
4- title: Login's title
 
5- content: login's content
 
6- img: URL of the image
 
7- draggable: true or false, enable or disable the draggable mode
 
8- Label1: First input label and place holder
 
9- Label2: Second input label and place holder
 
10- buttoncancel: Cancel's button label
 
11- buttonunlock: Unlock's button label
 
12- animation: Animation (see the animation list in the "Animation" section)
 

 

Side Panels

Side panels! with HTML capability

 

Properties

 

1- html: Sidepanel's HTML content
 
2- width: SidePanel's Width
 
3- position: "left" or "rigth" position
 
4- blackscreen: true or false, if you want to enabled or disabled the background
 
5- title: Sidepanel's title, this value will be visible on the notification bar (if is enabled)
 
6- content: Sidepanel's content. this value will be visible on the notification bar (if is enabled)
 
7- img: Sidepanel's image for the notification bar
 
8- notificationbar: true or false, if is enabled, this will appear in the notification panel
 
9- closebuttons: true or false, shows or hide the button section
 
10- iframe: iFrame URL (ONLY THE URL)
 
11- iframescrolling: true or false, enabled or disabled the iFrame Scrolling functionallity
 
12- animation: Animation Name (see animation section for more information)
 
13- timeout: Time in miliseconds for autohide.
 
 

Input Boxes

Stylish input fields!

 

Properties

1- type: "text", "password" or "select" are the valid options
 
2- selectvalues: Array of options, see tutorials for more information
 
3- withheader: true or false, if you want the header section
 
4- headertext: header's text (visible only if  "withheader" is enabled)
 
5- width: input's width
 
6- title: Input's title
 
7- content: Input's content
 
8- img: Image URL
 
9- draggable: true or false, enabled or disabled the draggable functionallity
 
10- placeholder: Input's place holder.
 
11- buttons: Array of buttons, see the tutorial for more information.
 
12- animation: In Animation (see "animation" section for more information)
 

 

MiniBoxes

 

Need loadings of special boxes?

 

Properties

1- content: Minibox's content
 
2- cssloading: true or false, if you want to use a CSS predefined loading
 
3- blackscreen: true or false, if you want to disabled the background
 
4- img: Image URL, this will never appear if the CSSLOADING property is enabled
 
5- blink: true or false, if you want to fadein and fadeout the content
 
6- clicktoclose: true or false, if you want to close when the user clicks the minibox
 
7- timeout: MInibox timeout in miliseconds to autoclose.

Special functions

From any part of your site, this is a Javascript, you can call this functions.

 

mpCloseMiniBoxes

mpCloseMiniBoxes();

This function closes ALL Miniboxes at the same time.

 

mpCloseAllModals

mpCloseAllModals();

This function closes all Modals at the same time.

 

mpOpenCloseBar(time)

mpOpenCloseBar(300)

Close all Sidebars in 300 milliseconds

 

SetNotificationBackground

SetNotificationBackground(URL);
